# Containerization-of-Java-project-using-Docker

## Why containerization is important :

#### 1. Consumes low resources - We can save lot of money
#### 2. Best suitable for the micro-service architecture
#### 3. Same containers on different environment - Reusable and reputable

## The project setup is done in this order 

#### 1. MySQL : Databse service  
#### 2. Memcache : DB chaching service 
#### 3. RabbitMQ : Queue service 
#### 4. Tomcat : Application service : We need to upload our artifact in the tomcat using maven which indeed need JDK 
#### 5. Nginx : Web service
